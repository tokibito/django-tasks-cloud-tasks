Simulate Task Execution

Cloud Tasks will send an HTTP POST request to your endpoint. For local testing without emulator, you can manually trigger the task:

curl -X POST http://localhost:8000/tasks/execute/ \
  -H "Content-Type: application/json" \
  -d '{
    "task_id": "{{ task_id }}",
    "task_path": "{{ task_path }}",
    "args": {{ args|default:"[]" }},
    "kwargs": {{ kwargs|default:"{}" }},
    "queue_name": "{{ queue_name|default:"default" }}",
    "backend": "default",
    "priority": 0,
    "takes_context": false,
    "enqueued_at": "2024-01-01T00:00:00+00:00"
  }'
